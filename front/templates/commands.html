Pins

You use the direction_pin and gas_pin to set the pin number on the Pi Hat.

Direction Pin
Gas Pin
Steering

left, straight and right parameters are used to send the direction order. You need to tune those values so that the Raspi sends the right PWM commands to the RC Car.

The invert_dir parameters defines the same commands but for reverse.

Gas

stop, neutral and drive are used to accelerate the car. You need to tune these parameters.

drive_max is the maximum amount of gas. Be careful, on some cars, huge values can lead to a very fast (and uncontrollable) car!
